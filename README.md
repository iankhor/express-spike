### Instructions

1. run `yarn install`
2. run `yarn db:migrate`
3. run `yarn db:seed:all`
2. to start app, run `yarn start:dev`
3. go to `localhost:3000`
4. observe amount in returned payload changing
5. to run tests, run `yarn jest`
